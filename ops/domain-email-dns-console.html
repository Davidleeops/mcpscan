After DNS Saves

  1. Run the approved launch-link command after Stripe links exist so landing/CNAME becomes mcpscanhq.com.
  2. In GitHub Pages settings, set the custom domain to mcpscanhq.com.
  3. Wait for DNS checks, then enable or confirm HTTPS enforcement.
  4. Verify https://mcpscanhq.com, https://mcpscanhq.com/secure-intake.html, and hello@mcpscanhq.com.

Do Not Skip

Do not publish outbound emails from hello@mcpscanhq.com until MX, SPF, and DKIM pass. If DKIM is not configured, deliverability will be weaker and security buyers may notice.

Verification Commands

Use these after DNS has had time to propagate.

dig +short mcpscanhq.com A
dig +short www.mcpscanhq.com CNAME
dig +short mcpscanhq.com MX

curl -L -I https://mcpscanhq.com/
