After Stripe Creates The Three Links

Paste the generated Payment Link URLs into this command. Keep the placeholders until the real Stripe links exist.

Open command builder 
npm run launch:apply-links -- --domain mcpscanhq.com --email hello@mcpscanhq.com --quick QUICK_PAYMENT_LINK --launch LAUNCH_PAYMENT_LINK --enterprise ENTERPRISE_PAYMENT_LINK
