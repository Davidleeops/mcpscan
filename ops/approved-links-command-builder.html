Approved Links Command Builder

Paste the approved domain, mailbox, and three Stripe Payment Links. This builds the exact command that updates the landing page, contact email, custom-domain file, and policy URLs.

Before You Build

  • The domain and mailbox should already be purchased.
  • The Stripe links should already exist in Stripe.
  • Do not use test links on the public landing page unless you intentionally want a test launch.

Inputs

Command

Fill the inputs, then build the command.

After Running

Commit/publish the changed landing files, then verify the custom domain, checkout buttons, thank-you page, secure intake page, and policy links.
