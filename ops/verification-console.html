Verification Console

Use this after each launch gate. It checks required local artifacts and public URLs, then reports warnings for founder-gated items that are not done yet.

Current GitHub Pages Verification

npm run launch:verify

Custom Domain Verification

Run after DNS and GitHub Pages custom domain are configured.

npm run launch:verify -- --domain mcpscanhq.com

Strict Mode

Use strict mode only after domain, email, Stripe links, and custom URLs are fully approved. It exits non-zero on warnings.

npm run launch:verify -- --domain mcpscanhq.com --strict
