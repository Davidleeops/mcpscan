# MCPScan Security Readiness Assessment

MCPScan is a fixed-scope readiness assessment for teams connecting AI agents to
MCP servers, internal tools, credentials, code, customer data, and third-party
APIs.

## What It Answers

- Which MCP servers and tools are in use?
- What can those tools access or change?
- Which credentials, files, APIs, data stores, tickets, docs, or deployment
  systems are reachable?
- Where are permissions, secrets, tool descriptions, or transport settings risky?
- What should be fixed before customer pilots, internal rollout, or enterprise
  security review?

## Included

- MCP server and tool inventory.
- Configuration and permission review.
- Secret exposure and sensitive path review.
- Prompt-injection and tool-description review.
- Risk-ranked findings.
- Remediation checklist.
- CI/readiness recommendation.

## Not Included

- Full penetration testing.
- Exploit development.
- Compliance certification.
- Legal opinion.
- Remediation implementation.
- Continuous monitoring unless separately agreed.

## Intake

Start with sanitized MCP configs, safe test targets, or high-level server/tool
inventories. Do not send raw production credentials, tokens, customer data, or
private configs through public issues or ordinary email.
